Edit client settings in client/client_settings.json. The file must parse as valid JSON.

Name: ClientStarcraftDir
Desc: Directory of Starcraft on Client machine
Allowed values: string; no spaces; End in "\", backslashes must be escaped.

Name: DefaultBWAPISettings
Desc: File of default BWAPI settings, relative to client.jar
Allowed values: string; no spaces; 

Name: TournamentModule
Desc: BWAPI Tournament Module Location, relative to Starcraft directory
Allowed values: string; no spaces; forward slashes allowed

Name: ServerAddress
Desc: IP address and port of server
Allowed values: string with format "[address]:[port]"
